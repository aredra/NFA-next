## 🚒 NFA(National Fire Agency)-next

`Next` 와 `tailwind` 를 사용한 2018년도 경기도 소방청 구조활동현황을 기반으로한 기초 프로젝트입니다.

자료출처 : [공공데이터포털](https://www.data.go.kr/data/15086458/fileData.do)

## 🔥 Getting Started

```sh
npm install
npm run dev
# or
yarn install
yarn dev
```
